## 💻 必要なもの

このプロジェクトを動かすには、以下がインストールされている必要があります：

- **Node.js** (バージョン 18 以上)
  - [公式サイト](https://nodejs.org/)からダウンロードできます
  - インストールされているか確認: `node --version`

## 🚀 インストール方法

### 1. プロジェクトのフォルダに移動

ターミナル（コマンドプロンプト）を開いて、このフォルダに移動します：

```bash
cd "このフォルダへのパス"
```

### 2. 必要なパッケージをインストール

以下のコマンドを実行します：

```bash
npm install
```

これで、開発に必要なツール（Vite）が自動でインストールされます。

## 🎮 使い方

### 開発サーバーを起動する

以下のコマンドを実行すると、ブラウザが自動で開きます：

```bash
npm run dev
```

成功すると：
- ブラウザが自動で開きます（`http://localhost:3000`）
- トップページが表示され、すべてのHTMLページにアクセスできます
- ファイルを編集すると、自動でブラウザが更新されます

### サーバーを停止する

ターミナルで `Ctrl + C` を押します。

### 本番用にビルドする

以下のコマンドで、本番環境用のファイルを生成できます：

```bash
npm run build
```

ビルドされたファイルは `dist` フォルダに出力されます。

### ビルドしたファイルをプレビューする

```bash
npm run preview
```

## 🔧 よくある質問

### Q: `npm install` で失敗する

**A:** Node.jsが正しくインストールされているか確認してください：
```bash
node --version
npm --version
```

### Q: ポート3000が使えない

**A:** 他のアプリケーションがポート3000を使用している可能性があります。
`vite.config.js` の `port: 3000` を別の番号（例: `port: 3001`）に変更してください。

### Q: 画像やJSONファイルが読み込めない

**A:** Viteは相対パスを自動で解決します。ファイルパスが正しいか確認してください。

## 📝 開発のヒント

1. **検索機能を使う**: トップページの検索ボックスで、ページ名やキーワードで絞り込めます
2. **直接アクセス**: `http://localhost:3000/eagles/index-bundled.html` のように直接URLを入力できます
3. **複数タブで開く**: 複数のページを同時に開発・確認できます
